# 3D Parallax Effect using HTML, CSS, and JavaScript

## Overview
This project demonstrates a smooth and interactive 3D Parallax Effect built using HTML, CSS, and JavaScript. It creates a visually engaging depth illusion where elements move at different speeds based on user interaction (mouse movement or scrolling), giving a realistic 3D experience.

---

## Features
- Real-time mouse-based 3D interaction  
- Layered parallax depth effect  
- Smooth animations with CSS and JavaScript  
- Responsive design (works on different screen sizes)  
- Clean and modern UI  

---

## Tech Stack
- HTML5 – Structure of the webpage  
- CSS3 – Styling and 3D transformations  
- JavaScript (Vanilla JS) – Interactivity and motion logic  

---

## Project Structure
3D-Parallax-effect/
│── index.html # Main HTML file
│── style.css # Styling and 3D effects
│── script.js # JavaScript for interaction
│── assets/ # Images or layers 

---

---

## How It Works
- Multiple layers are stacked using CSS properties like `transform`, `perspective`, and `translateZ`.
- JavaScript tracks mouse movement.
- Based on cursor position, each layer moves at a different speed.
- This difference in motion creates a depth illusion (parallax effect).
